## Theme: Prologue 

Prologue is a simple, single page responsive site template from [HTML5 UP](https://html5up.net/prologue), now available as a blog-aware Jekyll theme from [Chris Bobbe](https://chrisbobbe.github.io). It features a clean, minimalistic design and a sticky sidebar with navigation-linked scrolling.

**Original Theme Demo**: https://chrisbobbe.github.io/jekyll-theme-prologue/
